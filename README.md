# BogusAssistance - Templates & Web Apps

Welcome to BogusAssistance! This website provides a collection of templates, PDFs, Excel files, and web applications to help you with budgeting, project management, and more.

## 📁 Project Structure

```
BogusAssistance/
├── index.html          # Main website page
├── styles.css          # Website styling
├── README.md           # This file
├── templates/          # Folder for template files
│   ├── budgeting/      # Budgeting templates
│   ├── projects/       # Project management templates
│   └── other/          # Other templates
├── apps/               # Folder for web applications
│   ├── budget-app/     # Budget calculator web app
│   └── task-app/       # Task management web app
└── docs/               # Documentation
```

## 🚀 Features

- **PDF Templates** - Download ready-to-print documents
- **Excel Templates** - Download spreadsheets with formulas
- **Web Apps** - Interactive tools you can use in your browser
- **Easy Navigation** - Simple, clean interface organized by category

## 📝 How to Add Templates

1. Create folders in the `templates/` directory for your categories
2. Add your Excel, PDF, and Word files
3. Update the `index.html` to add new template cards
4. Link the download buttons to your files

## 🌐 Hosting on GitHub Pages

This site is hosted on GitHub Pages and will be automatically deployed when you push changes to the main branch.

**Access your site at:** `https://BogusQuade.github.io/BogusAssistance/`

### Enable GitHub Pages:
1. Go to your repository settings
2. Scroll down to "GitHub Pages"
3. Select "main" branch as the source
4. Save!
